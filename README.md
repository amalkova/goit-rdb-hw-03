# goit-rdb-hw-03

Домашнє завдання з курсу баз даних (MySQL).

## Що зроблено

- Імпортовано надані CSV-файли в базу `mydb`.
- Створено таблиці `products`, `shippers` та інші з датасету.
- Написано SQL-запити для завдань 1–5:
  - вибірка стовпчиків із таблиць `products` та `shippers`;
  - обчислення середньої, мінімальної та максимальної ціни;
  - вибір унікальних пар `category_id` + `price` з сортуванням і `LIMIT`;
  - підрахунок кількості продуктів у діапазоні цін;
  - агрегація за `supplier_id` (кількість продуктів і середня ціна).
- Додано файл `queries.sql` з усіма запитами та скріншоти виконання в MySQL Workbench.
